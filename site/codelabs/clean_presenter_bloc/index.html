• Presenter : Creates Screen based on BuildContext, BLoC, and ViewModel
  • BLoC :

Feature Widgetis a StatelessWidget that describes the feature user interface by initializing the BLoc and associate a presenter to the BLoC.

Presenter creates three different state of the screen such as Loading Screen, Error Screen and Screen, also manages all screen actions

    }

getViewModelStream Method : BLoc listener handle view model changes

buildLoadingScreen Method : Creat the loading screen

buildErrorScreen Method : Creat error screen

buildScreen Method : Creat the screen by passing associated viewModel

      @override
      DepositCheckCardScreen buildScreen(BuildContext   context,
          ExampleBloc bloc, ExampleViewModel  viewModel) {
        return ExampleScreen(
          viewModel: viewModel,
          pressenterActions: ExamplePressenterActions(),
        );
      }
